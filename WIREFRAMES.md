```

  â— Green    - Running, Success, Online
  â— Yellow   - Starting, Warning, Pending
  â— Red      - Error, Stopped (destructive action)
  â— Gray     - Inactive, Disabled, Offline
  â— Blue     - Information, Links, Primary actions

  Purple bg  - Bot's own messages
  Light gray - Other users' messages
  White      - System messages

  Green      - Active status (Running, Speaking)
  Gray       - Inactive status (Stopped, Listening)
  Blue       - Info badges (Room count, Message count)
  Yellow     - Warning badges
```
